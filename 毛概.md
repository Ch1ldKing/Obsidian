# 一、毛泽东思想及历史地位
## 1.毛泽东思想历史条件
### 主观条件
1. 思想上：陈独秀，胡适创办《新青年》
2. 组织上：创办新民学会，以革新学术，砥砺品行
3. 实践上：游学，工人夜校，领导学生运动
### 客观条件
1. 近代中国社会的特殊国情和中国革命的特殊性
2. 时代背景：19世纪末20世纪初，世界进入帝国主义和无产阶级革命时代，战争与革命成为时代主题
3. 实践背景：中国共产党领导人民进行革命和建设的成功实践是毛泽东思想形成和发展的实践基础
## 2.毛泽东思想形成过程
1. 萌芽阶段 1921-1927 （大革命时期）第一次国内革命战争结束 代表作《中国社会各阶级的分析》《湖南农民运动考察报告》
2. 思想形成 1927-1935 土地革命时期《中国的红色政权为什么能够存在？》《井冈山斗争》《星星之火可以燎原》《反对本本主义》

>提出了农村包围城市，武装夺取政权，标志着毛泽东思想的初步形成

3. 思想成熟 1935-1945 抗日战争时期 遵义会议 《实践论》《矛盾论》《论联合政府》

> 遵义会议确立了毛泽东同志在党中央和红军的领导地位 ；毛泽东系统地阐述了新民主主义革命理论，实现了马克思主义和中国革命实践相结合的历史性飞跃，标志着毛泽东思想得到多方面展开而趋于成熟

4. 写入党章 1945 七大通过的《中国共产党党章》将毛泽东思想确立为党必须长期坚持的指导思想
5. 继续发展：解放战争和新中国成立后《在中国共产党第七届中央委员会第二次全体会议上的报告》《论人民民主专政》《集中优势兵力，各个歼灭敌人》
## 3.毛泽东思想成熟原因
1. 实践基础：长征/北伐胜利；大革命，反围剿失败
2. 以毛泽东为代表的政治路线在全党地位的确立
3. 现实基础： 抗日战争的复杂环境和丰富实践
4. 全党思想路线的端正和理论素养的提高
## 4.毛泽东思想科学内涵

2. 是被实践了的 关于中国革命与建设的 正确的理论准则和经验总结：主要阐释了毛泽东思想与中国革命和建设实践的关系
3. 是中国共产党集体智慧的结晶：主要阐释了毛泽东思想的主要创立人毛泽东同志和中国共产党的关系

# 二、新民主主义革命理论
## 1.形成依据
1. 社会性质：中国逐步成为半殖民地半封建社会，这是近代中国最基本的国情
2. 主要矛盾：是帝国主义和中华民族的矛盾（主要矛盾）、封建主义和人民大众的矛盾
3. 革命性质：是一场传播新思想新文化新知识的伟大思想启蒙运动和新文化运动
4. 实践基础：
	+ 主观诉求：旧民主主义革命失败呼唤新的革命理论
	+ 客观基础：新民主主义革命的艰辛探索奠定了革命理论形成的实践基础
## 2.时代特征
1. 外在特征：俄国十月革命，标志中国资产阶级民主革命的<u>时代背景</u>发生根本转变
2. 自身特征：五四运动，标志中国无产阶级开始作为独立的政治力量登上历史舞台，中国革命进入新民主主义革命阶段
>由此（中共二大）中国革命两步走
>1. 完成反帝反封建的任务
>2. 完成社会主义革命任务

## 3.新民主主义革命的总路线
1. 新民主主义革命的对象
	+ 主要敌人：帝国主义、封建主义、官僚资本主义（三座大山）
		1. 帝国主义：首要对象，最大障碍，总根源
		2. 封建地主阶级：重要对象
		3. 官僚资本主义：依靠帝国主义、勾结封建势力、利用国家政权力量
2. 新民主主义革命的动力
	+ 无产阶级、农民阶级、城市小资产阶级和民族资产阶级
		1. 无产阶级：中国革命最基本的动力，新的社会生产力的代表，近代中国最进步的阶级
		2. 农民阶级：中国革命的主力军，中国革命战争就是党领导下的农民战争
		3. 城市小资产阶级：无产阶级的可靠同盟者，知识分子、小商人、手工业者、自由职业者
		4. 民族资产阶级：动力之一，但有动摇性
3. 新民主主义革命的领导力量
	1. 无产阶级的领导权
		 + 无产阶级的领导权是中国革命的中心问题。区别新旧民主革命的根本标志：领导权在无产阶级还是资产阶级手里
		 + 无产阶级及其政党的领导，是中国革命取得胜利的根本保证
	2. 无产阶级的优点特点
		 + 无产阶级从诞生起，就受外国资本主义、本国封建势力和资产阶级压迫，比其他阶级坚决彻底
		 + 集中分布在少数大城市和大企业，有利于组织和团结，有利于革命思想的传播和强大革命力量的形成
		 + 成员大部分出身于破产农民，和农民有着天然的联系，便于和农民结成亲密的联盟
	3. 实现无产阶级领导权
		*与资产阶级争夺领导权的斗争中实现*
		1. 必须建立以工农联盟为基础的广泛的统一战线，实现领导权的关键
		2. 必须建立巩固的工农联盟
		3. 坚持独立自主的原则
		4. 建立和发展人民的武装力量
	4. 新民主主义革命的性质和前途
		+ 近代中国 半殖民地半封建社会的性质 和 中国革命的历史任务，决定了新民主主义革命的性质**不是**无产阶级社会主义革命，而是资产阶级民主主义革命
		+ 要建立的是无产阶级领导的各革命阶级的**联合专政**，不是无产阶级专政
		+ 新民主主义革命“新”的特点
			1. 新民主主义革命处于世界无产阶级社会主义革命的时代，是世界无产阶级社会主义革命的一部分
			2. 革命的领导力量是中国无产阶级及其先锋队---中国共产党
			3. 革命的指导思想是马列主义
			4. 革命的前途是社会主义，不是资本主义
## 4.新民主主义的基本纲领
1. 政治纲领
	+ 推翻帝国主义和封建主义的统治，建立一个无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义的共和国
	+ 国体--各革命阶级联合专政 政体--民主集中制的人民代表大会制度
2. 经济纲领
	1. 没收封建地主阶级的土地归农民所有，是新民主主义革命**主要内容**
		+ 土地革命路线，依靠贫雇农（基本力量），团结中农，消灭封建剥削制度
	2. 没收官僚资产阶级的垄断资本归新民主主义的国家所有
		+ 反对官僚资本主义原因
			1. 同外国帝国主义、本国地主阶级、旧式富农密切结合，是买办的封建的国家垄断资本主义
			2. 同大地主资产阶级的反革命国家政权密切结合，成为国民党统治的经济基础
			3. 严重阻碍中国经济发展和社会进步
	3. 保护民族工商业
		+  由落后的生产力和新民主主义革命的性质所决定
		+ 先进的生产方式和经济成分，对发展现代技术和社会生产力有积极作用
		+ 让私人资本主义经济在不能操作国计民生的范围内活的发展的便利
3. 文化纲领
	+  新民主主义文化，就是无产阶级领导人民大众的反帝反封建的文化
		1. 民族的：反帝反压迫
		2. 科学的：反对封建迷信
		3. 大众的：为人民大众服务，武装人民大众，思想武器

### 4.发展战略理论
1. 三步走基本实现现代化
	1. 到1990，实现国民生产总值比1980翻一番，解决人民温饱问题
	2. 到20世纪末，使国民生产总值再增长一倍，人民生活达到小康水平
	3. 到21世纪中叶，人均国民生产总值达到中等发达国家水平，人民生活比较富裕，基本实现现代化，然后在这个基础上继续前进

	1. 能源交通：是我国国民经济的突出薄弱环节

### 5.社会主义改革开放理论

2. 改革是一场新的革命，是社会主义制度的自我完善和发展
3. 改革是社会主义社会发展的直接动力
4. 改革是一项崭新的事业，是一个大试验
5. 开放也是改革，对外开放是基本国策
	对外开放的要点
		1. 正确对待资本主义社会创造的现代文明成果
		2. 在对外开放过程中要高度珍惜并坚决维护中国人民经过长期奋斗得来的独立自主
### 6.社会主义市场经济理论
1. 过程
	1. 1981 计划经济为主、市场调节为辅
	2. 1984 公有制基础上有计划的商品经济
	3. 1992 南方谈话：计划多一点和市场多一点，不是社会主义与资本主义的本质区别
		*标志着邓小平社会主义市场经济理论的形成*
2. 内涵：
	1. 计划经济和市场经济不是划分社会制度的标志
	2. 计划和市场都是经济手段，对经济活动的调节各有优劣，社会主义实行市场经济是要把两者的优势结合起来
	3. 市场经济作为资源配置手段本身不具有制度属性，可以和不同的社会制度结合，从而表现出不同的性质
